보존 기준점 참조 속성 기능 추가 완료
====================================

[구현 내용]

1. 보존 기준점 업로드 시 점번호 정보 저장
   - Excel: 3열에 점번호 (없으면 행번호 사용)
   - Shapefile: ID, FID, OBJECTID, 점번호 등 속성 자동 인식

2. 중복 분석 시 속성 정보 추가
   - AI 점과 20m 이내 일치 시: "보존기준점(번호)와 일치"
   - 사용자 추가 점이 20m 이내: "보존기준점(번호) 참고"
   - 거리 정보도 함께 저장

3. Canvas에서 사용자 점 추가 시
   - 20m 이내 보존 기준점 자동 감지
   - 속성 정보 자동 추가

4. 내보내기 시 속성 컬럼 추가
   - 비고: 보존 기준점 참조 정보
   - 참조ID: 보존 기준점 번호
   - 거리: 보존 기준점까지 거리

[테스트 방법]

1. 프로그램 실행 후 지구계 파일 업로드
2. AI 분석 실행하여 기준점 검출
3. 보존 지적기준점 Excel/SHP 파일 업로드
   - Excel: Y(1열), X(2열), 점번호(3열)
   - SHP: Point 타입, ID 속성 포함
4. 자동으로 20m 반경 중첩분석 수행
5. 화면에서 추가로 점 편집
6. SHP/GPKG로 내보내기
7. 속성 테이블에서 "비고" 컬럼 확인

[활용 방법]

- AI가 놓친 보존 기준점 위치에 수동으로 점 추가
- 내보낸 파일의 속성 정보로 검증 및 추적 가능
- 보존 기준점과의 일치/참고 여부 명확히 구분
